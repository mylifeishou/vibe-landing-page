# 바이브 랜딩페이지

신뢰 기반 부동산 매칭 플랫폼 "바이브"의 랜딩페이지입니다.

## 배포

이 페이지는 GitHub Pages를 통해 자동으로 배포됩니다. `main` 브랜치에 푸시할 때마다 자동으로 배포됩니다.

## 로컬에서 실행

1. 파일을 다운로드합니다
2. `index.html` 파일을 브라우저에서 열거나 로컬 서버를 실행합니다:

```bash
# Python 3
python -m http.server 8000

# Node.js (http-server)
npx http-server
```

그 후 브라우저에서 `http://localhost:8000`으로 접속하세요.
